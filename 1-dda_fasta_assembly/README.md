DDA FASTA Assembly
Purpose
amyloid-dda-fasta.py constructs a proteomics search database used for data‑dependent acquisition (DDA) analysis of tissue amyloid deposits. The script assembles a FASTA file containing:

canonical human protein sequences
selected amyloid‑associated protein isoforms
missense variants derived from UniProt Humsavar
supplemental peptide or protein targets
The resulting database is intended for LC–MS/MS peptide identification in amyloid proteomics workflows.

Script
amyloid-dda-fasta.py

Required Inputs
The script expects the following files in the working directory.

UP000005640_9606.fasta
Canonical human reference proteome from UniProt.

Download from:
https://www.uniprot.org/proteomes/UP000005640

uniprot_sprot_varsplic.fasta
Swiss‑Prot isoform FASTA from UniProt containing curated protein isoforms.

Download from:
https://ftp.uniprot.org/pub/databases/uniprot/current_release/knowledgebase/complete/uniprot_sprot_varsplic.fasta.gz

humsavar.txt
UniProt Humsavar table containing curated human missense variants.

Download from:
https://ftp.uniprot.org/pub/databases/uniprot/current_release/knowledgebase/variants/humsavar.txt

protein_ids.txt
List of UniProt accessions corresponding to amyloid‑associated proteins analyzed in this study.

supp_targets.fasta
Supplemental peptide or protein targets (for example therapeutic peptides or other non‑canonical amyloidogenic sequences).

Example Directory Structure
The script assumes all input files are located in the same working directory.

Example:

1-dda_fasta_assembly/

amyloid-dda-fasta.py
protein_ids.txt
supp_targets.fasta
humsavar.txt
UP000005640_9606.fasta
uniprot_sprot_varsplic.fasta

Output
The script creates an output/ directory containing:

per‑protein FASTA files for canonical sequences, isoforms, and variants
intermediate variant tables
a combined FASTA database
output/amyloid_dda.fasta

This FASTA contains:

the full canonical human proteome
supplemental targets
isoform and variant sequences for proteins listed in protein_ids.txt
Run
From the directory containing the input files:

python amyloid-dda-fasta.py

Notes
Variant sequences are generated by applying missense substitutions from humsavar.txt to canonical protein sequences.

Isoform and variant sequences are appended to the combined FASTA database to ensure detection of clinically relevant sequence variants during database searching.

The script assumes that all input files are located in the current working directory.
